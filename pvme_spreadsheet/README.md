# Development

## Install

```commandline
cd pvme-settings\<sub_folder>
```
```commandline
virtualenv venv
venv\Scripts\activate
pip install pip-tools
pip install python-dotenv
```
*Note: `python-dotenv` installed separately to avoid `dev-requirements.in`. Save as it has no dependencies*
